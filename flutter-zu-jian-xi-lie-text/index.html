flutter组件系列-Text

2019-12-03 flutter

瞅瞅text组件长啥样跟该怎么用。

Text

在flutter中没有像我们平时用的那么多关于文字的标签，统一是用Text组件来渲染文字，先来看代码实现：

const Text(
    this.data,  //要展示的文字
		{
    Key key,
    this.style,  //文字的样式
    this.strutStyle,  //文字的支撑样式
    this.textAlign,  //文字的对齐方式
    this.textDirection, //文字方向
    this.locale,
    this.softWrap,  //是否自动断行
    this.overflow,   //超出样式
    this.textScaleFactor,  //文字缩放倍数
    this.maxLines, //最大行数
    this.semanticsLabel,
    this.textWidthBasis,
  })

使用flutter的Text组件十分方便，可以通过设置不同的参数来展示不同的形式而不受排版布局的影响。

一个列子🌰

Text(
            'Hello WorldHello WorldHello WorldHello World',
            style: TextStyle(
              color: Colors.black,
              fontSize: 32,
              fontWeight: FontWeight.bold,
              letterSpacing: 1,
              wordSpacing: 1,
            ),
            textAlign: TextAlign.center,
            softWrap: true,
            overflow: TextOverflow.ellipsis,
            textScaleFactor: 1,
            textDirection: TextDirection.ltr,
            maxLines: 2,
        ),
